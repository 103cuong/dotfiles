## Install

Clone this project:
```
$ git clone https://github.com/cuongw/dotfiles.git
```
cd `dotfiles`:
```
$ cd dotfiles
```
Allow execute shell script:
```
$ chmod +x install.sh
```
Run `install.sh` file:
```
$ yes | ./install.sh
```

Enjoy 😘
