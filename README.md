## Install

### Step 1: Clone this project

```sh
$ ghttps://github.com/cuongw/dotfiles.git && cd dotfiles
```

### Step 2: Allow execute shell script

```sh
$ chmod +x install.sh
```

### Step 3: Run `install.sh` file

```sh
$ yes | ./install.sh
```
