## Install

### Step 1: Clone this project

```
$ git clone https://github.com/jade28/dotfiles.git && cd dotfiles
```

### Step 2: Allow execute shell script

```
$ chmod +x install.sh
```

### Step 3: Run `install.sh` file

```
$ yes | ./install.sh
```
